# Portfolio de Warlley

Este é um modelo simples de portfólio pessoal em HTML/CSS/JS.

- Abra `index.html` no navegador para visualizar.
- Edite o conteúdo em `index.html`, as cores e layout em `styles.css` e comportamentos em `script.js`.

Sugestões:
- Substitua a imagem de perfil por `assets/profile.jpg` (pasta `assets` não incluída).
- Atualize a seção `Projetos` com links reais para seus repositórios.
